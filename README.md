# Movieapp
This Flutter application provides information regarding the latest, trending, and top-rated movies and TV shows. Users can view ratings, release dates, posters, and detailed information about films and shows. Additionally, users can search for their favorite movies.

## Features

- **Firebase Authentication**
  - Sign up, sign in, and reset password options are integrated using Firebase Authentication.
  - Users can securely sign in to access their personalized data.

- **TMDB API Integration**
  - The app fetches movie and TV show data from the TMDB API to provide the latest, trending, and top-rated content.

- **Home Page**
  - Displays three rows:
    - Trending Movies & Shows
    - Top Rated Movies
    - Now Playing
  - Users can click on the movie name or poster to view detailed information.

- **Search Bar**
  - A search bar is available on the home page for users to search for specific movies or TV shows.
  - The search results show a list of matching titles, and users can click on any movie to view its description.

- **Description Page**
  - Contains detailed information about a movie or TV show:
    - Release Date
    - Rating
    - Overview
